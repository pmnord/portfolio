Concurrent Kanban Board

A Kanban style issue tracker implemented with React on the frontend and with Node.js and PostgreSQL on the backend. Supports concurrent users with a drag-and-drop interface.

This project required normalizing state between the server and client and extensive work with third party libraries. The interface went through several iterations, during which I learned a lot of about common UI elements and patterns.

Plant Research App

This is a full stack app with a user account system using JSON Web Tokens for authorization.

This project involved learning about authorization and authentication systems, as well as data caching strategies on the server to minimize the amount of calls to the third party API.

I also deployed a Server Side Rendered version as a way to explore SSR with basic React.
